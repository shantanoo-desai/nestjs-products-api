# NESTJS-Products-API

Academind's Video Tutorial on making RESTful API __Nest__ with __MongoDB__.

This repository will work on integrating Tests with Travis-CI and Docker.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
